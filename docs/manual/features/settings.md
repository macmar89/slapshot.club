# Nastavenia (Settings)

Sekcia Nastavenia umožňuje používateľom prispôsobiť si svoj profil a správanie aplikácie.

## Možnosti nastavenia

- **Profil**: Zmena mena, prezývky a profilového obrázku.
- **Notifikácie**: Nastavenie upozornení na dôležité udalosti, ako je začiatok zápasu alebo nové výsledky.
- **Jazyk**: Prepínanie medzi podporovanými jazykmi (napr. slovenčina, angličtina).
- **Zabezpečenie**: Správa hesla a dvojfaktorovej autentifikácie.

## Odhlásenie

Tlačidlo na bezpečné odhlásenie z aplikácie sa nachádza v spodnej časti sekcie nastavení.
